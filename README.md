### Сборка и запуск практик

Каждая практика — отдельная программа. Пример (ПЗ№1):

```bash
cd practices/pz1-calculator
g++ -std=c++17 calculator.cpp -o calculator
./calculator
```

ПЗ№6 состоит из нескольких файлов:

```bash
cd practices/pz6-inheritance
g++ -std=c++17 main.cpp computer.cpp -o demo
./demo
```

## Курсовой проект «Продажа автомобилей»

Моделирование работы автосалона: учёт автомобилей (новых и подержанных),
менеджеров и клиентов, оформление сделок со скидками, статистика, сохранение
и загрузка данных в текстовый файл. Архитектура — паттерн **MVC**.

### Сборка и запуск

```bash
cd coursework-car-dealership
make
./dealership
```

Либо без `make`:

```bash
g++ -std=c++17 *.cpp -o dealership && ./dealership
```

Управление — через текстовое меню (ввод номера пункта и Enter). При первом
запуске загружается демонстрационный набор данных.

## Требования

- Компилятор C++ с поддержкой стандарта **C++17** (g++ / GCC, clang, MSVC).
- Утилита `make` — для сборки курсового одной командой (необязательно).
